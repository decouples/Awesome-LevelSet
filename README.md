# A list of Level Set codes!
# How to use
> Level-Set-01:
----
    For 2D demo, run from inside Matlab:
    >> run
    Select rectangular region (left click top left, drag and release at bottom right)
    For 3D demo, run from inside Matlab:
    >> run3
> Level-Set-02
----
    Just run the Main.py
